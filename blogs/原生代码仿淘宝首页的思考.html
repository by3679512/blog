    舟羽 2023/3/25 随笔

    # 前言

    仿淘宝主页是一个很好的练手项目，可以帮助我们熟悉HTML、CSS和JavaScript的使用,并且可以练习使用原生代码完成常见组件的实现。在这篇博客中，我将介绍一下如何使用原生代码实现仿淘宝主页。 淘宝首页虽然复杂，但拆分开主要由以下几个部分组成：

    • 顶部导航栏
    • 搜索框区域
    • 轮播图
    • 推荐商品区域
    • 热门商品区域
    • 底部信息栏

    # 主体内容部分

    主体内容的布局需要按块拆分，先拆分包括分类，导航，轮播图和个人信息相关部分

    总体思路是按照左右拆分，使用display：flex;将左右两部分变成盒模型，左边固定宽度为width: 300px;，右边部分铺满整个盒模型。接着在左边布局上，需要设置为盒模型，并且铺开按照每行三个子元素排列；关键代码为

    
/* 父元素样式 */
display: flex;
justify-content: center;
text-align: center;
flex-wrap: wrap; /* 子元素自动换行 */
/* 子元素样式 */
width: 33.33%; /* 一行放3个子元素 */

    最后即可实现左边布局。

    在右边布局上，需要先拆为上下两个部分，上部分为导航，下部分为轮播图和个人信息相关部分。 设置

    /* 父元素样式 */
display: flex;
flex-direction: column;/* 垂直排列 */
/* 子元素样式 */
flex:1;

    接着在下部分的轮播图和个人信息相关部分进行左右拆分

    /* 父元素样式 */
display: flex;
/* 子元素样式 */
flex: 1;
height: 380px;

    在这个布局的左边放置轮播图组件，再对右边个人信息相关部分进行上下拆分，设置 /* 父元素样式 / display: flex; flex-direction: column;/ 垂直排列 / / 子元素样式 */ flex:1; 最后即可实现布局 完整代码如下
